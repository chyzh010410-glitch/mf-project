# 通用客服 Agent V1 范围

## 一句话定义

通用客服 Agent 是苗丰智能服务体系的第一版 Agent，面向用户咨询场景，支持知识问答、工具查询、失败兜底和数据沉淀。

## 主要场景

1. 种植问题咨询
   - 苗木养护
   - 肥料使用
   - 常见病虫害
   - 季节性种植建议

2. 商品咨询
   - 查询商品
   - 商品适用场景
   - 商品详情解释

3. 百科问答
   - 查询树木百科
   - 解释百科内容

4. 订单咨询
   - 查询订单状态
   - 发货状态解释
   - 售后流程说明

5. 商家入驻咨询
   - 入驻条件
   - 审核流程
   - 入驻入口引导

## 兜底策略

当 Agent 无法回答时：

- 使用 `doubt` 状态追问澄清。
- 对明确无法解决的问题，写入未解决问题池。
- 给出人工处理或后续联系提示。

## 数据沉淀

每次咨询至少沉淀：

- 用户问题。
- Agent 回答。
- 命中的工具。
- 是否解决。
- 是否进入未解决问题池。
- 是否进入样本候选池。
