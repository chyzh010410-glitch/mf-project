  请求处理全流程（举个例子）

  用户登录时：
  浏览器 POST /client/auth/login {username, password}
      ↓
  ClientAuthController.login()          ← api层：接收参数
      ↓
  UserService (继承 MyBatis-Plus IService)
      ↓
  UserMapper (继承 MyBatis-Plus BaseMapper)
      ↓
  MySQL user表查询
      ↓
  返回 UserLoginResultVO {token, userId, username...}

  核心模式速查

  ┌────────────────┬───────────────────┬───────────────────────────────────────┐
  │   你要做的事   │    在哪个模块     │               文件示例                │
  ├────────────────┼───────────────────┼───────────────────────────────────────┤
  │ 定义数据表字段 │ common/entity/    │ User.java 继承 BaseEntity             │
  ├────────────────┼───────────────────┼───────────────────────────────────────┤
  │ 接收前端参数   │ common/dto/       │ UserLoginDTO.java 加 @NotBlank        │
  ├────────────────┼───────────────────┼───────────────────────────────────────┤
  │ 返回数据给前端 │ common/vo/        │ UserLoginResultVO.java                │
  ├────────────────┼───────────────────┼───────────────────────────────────────┤
  │ 数据库查询     │ core/mapper/      │ UserMapper.java 继承 BaseMapper<User> │
  ├────────────────┼───────────────────┼───────────────────────────────────────┤
  │ 写业务逻辑     │ core/serviceImpl/ │ UserServiceImpl.java                  │
  ├────────────────┼───────────────────┼───────────────────────────────────────┤
  │ 提供HTTP接口   │ api/controller/   │ /client/auth 下的 Controller          │
  └────────────────┴───────────────────┴───────────────────────────────────────┘

  关键约定

  - 返回值统一包装：所有接口返回 ResultVO<T>，成功 ResultVO.success(data)，失败 ResultVO.fail("消息")
  - 分页统一：入参 PageDTO (page, size)，出参 PageVO<T> (total, page, size, records)
  - 所有表都有：ID主键、createTime、updateTime、deleted逻辑删除
  - 不用写SQL：MyBatis-Plus的 lambdaQuery() 链式调用就能完成CRUD
  - 管理员路径 /admin/**，客户端路径 /client/**
  - 权限：JWT拦截器检查 login:token:xxx 或 client:token:xxx 是否在Redis中

  四、如何新增一个功能（以"添加公告管理"为例）

  后端步骤

  1. 建表 — 在 schema.sql 加：
  CREATE TABLE announcement (
      id BIGINT NOT NULL,
      title VARCHAR(200) NOT NULL,
      content TEXT,
      ...
      PRIMARY KEY (id)
  );

  2. 写实体 — common/entity/Announcement.java：
  @Data
  @EqualsAndHashCode(callSuper = true)
  @TableName("announcement")
  public class Announcement extends BaseEntity {
      private String title;
      private String content;
  }

  3. 写Mapper — core/mapper/AnnouncementMapper.java：
  @Mapper
  public interface AnnouncementMapper extends BaseMapper<Announcement> {}

  4. 写Service — core/service/AnnouncementService.java + core/serviceImpl/AnnouncementServiceImpl.java

  5. 写Controller — api/controller/client/ClientAnnouncementController.java

  前端步骤

  1. 写API — src/api/announcement.js：
  import request from '@/utils/request'
  export function getAnnouncements(params) {
    return request({ url: '/client/announcements', method: 'get', params })
  }

  2. 写页面 — src/views/announcement/index.vue

  3. 加路由 — src/router/index.js 加一条路由

  4. 构建验证 — npm run build
