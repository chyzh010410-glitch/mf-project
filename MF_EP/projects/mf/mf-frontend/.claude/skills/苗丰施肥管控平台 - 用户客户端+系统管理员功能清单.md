# 苗丰施肥管控平台 - 前端开发规范

## 固定技术栈（100% 严格遵守，不可替换）

Vue 3 (Composition API, `<script setup>`) + Vite + JavaScript + Element Plus + Axios + Vue Router + Pinia

## 设计规范

- **主题**: 白色 + 绿色 (`#2d8c4a` primary, `#f0f9f4` background, `#1b3a2a` sidebar)
- **布局**: 可折叠侧边栏 (220px -> 64px) + 顶部导航 + 主内容区
- **CSS**: 全局变量定义在 `src/styles/variables.css`，重置和工具类在 `src/styles/global.css`
- **HTTP**: `src/utils/request.js` 封装 Axios，baseURL `http://localhost:8080`，自动注入 Bearer token，401 重定向 /login
- **API 模块**: 每个后端控制器一个文件，放在 `src/api/`
- **状态管理**: Pinia，token 和 userInfo 持久化到 localStorage
- **路由守卫**: `router.beforeEach` 检查 token，未登录重定向 /login
